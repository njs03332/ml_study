3장에서 지도 학습 훈련 과정을 소개하고 2개의 예제를 다루어 봅니다. 첫 번째 예제는 작은 2차원 합성 데이터와 퍼셉트론을 사용하여 이진 분류를 수행합니다. 두 번째 예제는 옐프(Yelp) 리뷰 데이터셋과 퍼셉트론을 사용해 이진 분류 작업을 수행합니다.
